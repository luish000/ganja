GANJA

Ganja es un proyecto que permite a los usuarios controlar sus dispositivos moviles desde cualquier parte

Creado por nucleos.io / @nucleos

Funcionalidades?

  • Bloqueo o desbloqueo del dispositivo
  • Seguimiento en tiempo real
  • Recuperar o eliminar archivos
  • Ver historial de llamadas o SMS
  • Ver notificaciones
  • Activar o desactivar la camara
  • Realizar llamadas o enviar SMS

Una simple idea..

Para ser unicos en el mercado aparte de tener un panel administrativo para controlar los dispositivos, creo que podemos crear un BOT

Tecnologias a utilizar..

Creo que para adaptarnos a las nuevas tecnologias podemos utilizar como framework para nuestro front a React.js y para nuesto back a Sails.js
